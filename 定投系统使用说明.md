# 平衡型定投系统使用说明

## 概述

本系统基于**方案2：平衡型定投方案**，为年轻投资者提供自动化定投解决方案。

### 投资配置

- **总投入**: 3000元/月
- **交易佣金**: 万分之一（0.01%）
- **交易单位**: 100股整数倍

## 快速开始

### 方式1: Windows图形界面（推荐）

双击运行 `定投系统.bat`，根据菜单选择操作：

```
[1] 执行月度定投    - 手动触发定投
[2] 检查持仓状态    - 查看当前盈亏
[3] 显示完整状态    - 月度信息+持仓
[4] 导出交易历史    - CSV格式
[5] 显示交易日历    - 查看交易日
[6] 启动定时调度    - 自动运行
[7] 帮助信息        - 详细说明
[0] 退出
```

### 方式2: 命令行操作

```bash
# 进入目录
cd d:\temp\红利ETF

# 执行月度定投
python dca_main.py run

# 检查持仓
python dca_main.py check

# 查看完整状态
python dca_main.py status

# 导出数据
python dca_main.py export

# 启动定时调度
python dca_scheduler.py start
```

## 功能模块说明

### 1. dca_config.py - 配置模块
- 定义ETF组合配置
- 设置定投参数（金额、佣金、最小交易单位）

### 2. dca_calendar.py - 日历模块
- 获取交易日信息
- 判断是否为每月第一个交易日
- 生成交易日历

### 3. dca_trading.py - 交易模块
- 获取实时ETF价格
- 计算可购买股数（100的整数倍）
- 计算交易成本（佣金）
- 执行定投交易

### 4. dca_portfolio_manager.py - 持仓管理
- 记录持仓信息
- 管理交易历史
- 计算盈亏和收益率
- 导出CSV数据

### 5. dca_main.py - 主程序
- 整合所有模块
- 提供用户交互界面

### 6. dca_scheduler.py - 定时任务
- 每日09:25自动检查
- 每周五15:30生成周报
- 每月第一个交易日执行定投

## 使用场景

### 场景1: 首次设置

1. 双击 `定投系统.bat`
2. 选择 [7] 查看帮助，了解配置
3. 选择 [5] 查看交易日历
4. 在第一个交易日选择 [1] 执行定投

### 场景2: 日常查看

1. 双击 `定投系统.bat`
2. 选择 [2] 检查持仓状态
3. 查看当前盈亏情况

### 场景3: 自动运行

1. 双击 `定投系统.bat`
2. 选择 [6] 启动定时调度
3. 系统将在每个交易日自动执行

### 场景4: 数据分析

1. 定期选择 [4] 导出交易历史
2. 打开生成的CSV文件
3. 在Excel中进行更详细的分析

## 常见问题

### Q1: 行情获取失败怎么办？
**A**: 系统有多个备用数据源，如全部失败会显示提示，可稍后重试。

### Q2: 金额不足100股怎么办？
**A**: 系统会自动计算100股整数倍，可能实际投入少于计划金额。

### Q3: 如何修改每月定投金额？
**A**: 编辑 `dca_config.py` 中的 `DCA_CONFIG['monthly_amount']`

### Q4: 如何添加新的ETF？
**A**: 在 `dca_config.py` 的 `ETF_PORTFOLIO` 列表中添加新配置。

### Q5: 交易佣金如何计算？
**A**: 万分之一佣金，不足5元按5元收取。
