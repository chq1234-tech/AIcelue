# 平衡型定投系统 Web版 使用说明

## 快速开始

### 启动Web服务器

**双击运行**：`启动Web服务器.bat`

等待几秒钟，窗口会显示类似信息：
```
正在启动Web服务器...
请在浏览器中打开以下地址：
  http://127.0.0.1:5000
```

### 打开浏览器

1. 打开您的浏览器（Chrome、Edge、Firefox等）
2. 在地址栏输入：`http://127.0.0.1:5000`
3. 按回车键

### 访问Web界面

浏览器会显示定投系统的Web界面，包含：
- 首页（系统状态、快速操作）
- 持仓（当前持仓、收益统计）
- 配置（ETF配置详情）
- 日历（交易日信息）
- 历史（交易记录）
- 诊断（系统诊断）

## 功能说明

### 1. 首页
- 查看当前系统状态

## 使用流程

### 首次使用

1. **双击** `启动Web服务器.bat`
2. **打开浏览器** 访问 `http://127.0.0.1:5000`
3. **点击诊断** 检查系统环境是否正常
4. **查看配置** 确认ETF配置无误
5. **查看日历** 了解本月交易日
6. **执行定投** 在首交易日点击"执行月度定投"

### 日常使用

1. **双击** `启动Web服务器.bat`
2. **打开浏览器** 访问 `http://127.0.0.1:5000`
3. **查看首页** 了解系统状态
4. **查看持仓** 了解当前盈亏
5. **查看历史** 回顾交易记录

## 注意事项

### 端口占用
如果提示端口5000被占用，可能原因：
- 之前启动的服务器未关闭
- 其他程序占用了该端口

解决方法：
1. 关闭之前打开的命令行窗口
2. 或修改 `dca_web.py` 中的端口号（将 `port=5000` 改为 `port=5001`）

### 首次运行
首次启动可能需要安装依赖库，请耐心等待。如果安装失败，请手动运行：
```bash
pip install flask akshare pandas python-docx schedule chinese-calendar
```

### 网络要求
- 获取行情需要网络连接
- 节假日可能无法获取最新数据

## 故障排除

### 问题1：浏览器无法访问
- 确认服务器已启动（窗口应保持打开）
- 确认地址正确：`http://127.0.0.1:5000`
- 尝试使用：`http://localhost:5000`

### 问题2：页面显示错误
- 刷新浏览器页面（Ctrl+F5）
- 重启Web服务器（关闭后重新双击启动）

### 问题3：数据不更新
- 点击页面上的"刷新数据"按钮
- 或重新访问页面

### 问题4：执行定投失败
- 检查网络连接
- 查看诊断页面的错误信息
- 查看命令行的错误提示

## 快速参考

### 启动命令
```
双击: 启动Web服务器.bat
```

### 访问地址
```
http://127.0.0.1:5000
```

### 停止服务器
```
在启动窗口中按: Ctrl + C
```

### 修改端口（如需）
编辑 `dca_web.py`，找到：
```python
app.run(host='0.0.0.0', port=5000, debug=True)
```
将 `5000` 改为其他端口（如 `5001`）
