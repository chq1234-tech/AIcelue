# 平衡型定投系统 Web版 使用指南

## 快速启动

1. 双击 `启动Web服务器.bat`
2. 浏览器会自动打开 http://127.0.0.1:8080
3. 按 Ctrl+C 停止服务器

## 功能模块

| 页面 | 说明 |
|------|------|
| 首页 | 系统总览和统计 |
| 持仓 | 当前ETF持仓和盈亏 |
| 配置 | 定投策略参数设置 |
| 日历 | 定投日历和历史记录 |
| 历史 | 历史操作记录 |
| 诊断 | 系统状态检查 |

## 依赖

- Python 3.8+
- flask, akshare, pandas, python-docx, schedule, chinese-calendar
