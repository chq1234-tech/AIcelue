# 红利ETF定投系统 — 百分比再平衡策略

基于目标配比自动高抛低吸的 ETF 定投系统。

## 核心策略

**百分比再平衡定投（Rebalance DCA）**
- 每月固定注入 ¥3,000
- 以总资产为分母，按目标配比逐只计算每只 ETF 应持市值
- 超配的卖，低配的买——资金先卖后买循环使用
- 无论牛熊、每月必投，自动高抛低吸

## 快速开始

```bash
# 切换到百分比再平衡策略
python dca_main.py switch rebalance

# 执行本月定投
python dca_main.py run

# 查看系统状态
python dca_main.py check
```

## 策略切换

- `rebalance` — 百分比再平衡（推荐）
- `va` — 价值平均法
- `dca` — 普通定投

## 文件结构

| 文件 | 功能 |
|------|------|
| dca_config.py | ETF组合配置（配比、代码、月投金额） |
| dca_value_averaging.py | 策略核心模块（VA + 百分比再平衡） |
| dca_main.py | 主调度程序（策略选择、执行、报告） |
| dca_calendar.py | 交易日判断（含 chinese_calendar） |
| dca_portfolio_manager.py | 持仓管理、数据持久化 |
| dca_trading.py | 行情获取（多源降级） |
| dca_stop_profit.py | 止盈轮动（动态阈值） |

## 依赖

- Python 3.8+
- chinese-calendar（节假日判断）
- mootdx（通达信行情，可选降级）
